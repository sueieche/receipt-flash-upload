⚡ ReceiptFlash Worker

Cloudflare Worker 代碼，下載後放進你的 GitHub repo：

# 在你的電腦運行： git clone https://github.com/sueieche/ReceiptFlash.git cd ReceiptFlash # 複製以下文件到 repo 根目錄： # src/index.ts # wrangler.toml # package.json # tsconfig.json # .gitignore # README.md git add . git commit -m "feat: add upload worker" git push origin main

文件列表：

src/index.ts — Worker 主代碼，接收文件上傳到 R2
wrangler.toml — Worker 配置，綁定 R2 bucket
package.json — npm 依賴
tsconfig.json — TypeScript 配置
.gitignore — 忽略 node_modules 等
README.md — 使用說明

然後在 Cloudflare → Workers & Pages → Create application → Connect to GitHub → 選擇 ReceiptFlash repo → 部署完成後，Worker URL 就是你的上傳端點。
